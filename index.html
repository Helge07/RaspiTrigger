RasPiTrigger

 

Der RasPiTrigger ist eine Hardware/Software-Lösung zur Aufnahme von Wassertropfen-Skulpturen mit einem Raspberry Pi:

  • Das Programm ist eine JavaFX-Anwendung und wird auf dem Raspbery Pi installiert

  • In einer graphischen Benutzer-Oberfläche werden die Parameter für die Aufnahme eingestellt.

  • Das Programm übernimmt die Steuerung aller beteiligten Geräte (Kamera, Blitzgeräte und Magnetventile).

  • Die Geräte sind an den Raspberry Pi über eine Anschluss-Box angeschlossen, welche sie elektrisch von den GPIO-Ports des Raspberry Pi trennt.
